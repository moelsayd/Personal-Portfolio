# Personal Portfolio Website

## Overview
A sleek, responsive personal portfolio built with pure HTML and CSS—no frameworks, no extra dependencies. Designed for professionals and students in web, tech, or creative fields who want to showcase skills, projects, and contact info in style.

## Features
- Modern, animated UI
- Fixed transparent header with easy navigation
- Home section with name, role, and intro
- Highlighted CTA buttons (Hire Me | Let's Talk)
- Social media integration (GitHub, Facebook, LinkedIn)
- Responsive on mobile and desktop
- Custom styling, hover effects, fluid layouts
- Clean, maintainable code for easy updates

## Technologies Used
- HTML5
- CSS3 (Flexbox, transitions, keyframe animations)
- No JS dependencies—fast and lightweight

## Installation
1. Download and unzip the project files.
2. To run locally, open `index.html` in any browser.
3. To customize, edit content in `index.html` and styles in `style.css`.
4. Replace the `home.jpg` file for your own photo/banner as needed.
5. Deploy on any static hosting (Github Pages, Netlify, Vercel).

## Customization Guide
- Update name, bio, and links in `index.html`
- Change brand colors and fonts in `style.css`
- Add sections: About, Services, Portfolio, Contact (structure included)
- Easily connect to freelance/contact platforms
